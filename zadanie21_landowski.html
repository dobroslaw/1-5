Co znaczy skrót ID?

ID - identyfikator

Jaki jest cel stosowania ID?

ID jest stosowane w celu przypisania różnych cech do tych samych elementów/tagów

Jakie jest znaczenie znaku #

Za pomocą # definiujemy styl ID

Jak następuje wywołanie ID (wraz z wytłumaczeniem)?

id="nazwa ID"

Uwagi na temat stosowania ID (trzy)

  1. ID możemy możemy zastosować tylko 1 raz w całym dokumencie, a class nieskończoną ilość razy
  2. Podobnie jak klasy, identyfikatorów, możemy używać tyle razy, ile razy uznamy to za konieczne
  3. Elementy oznaczone ID muszą być unikatowe, tzn., że na jednej stronie może wystąpić tylko 1 element z takim ID
